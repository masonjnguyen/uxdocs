# Build problem statements

[Problem statement template](https://docs.google.com/presentation/u/3/d/1FEDYoW4ht6nfENBqMEjbOfSOEnEB4sD12JTjFE7Wff4/template/preview)  
[Hypothesis statement template](https://docs.google.com/presentation/d/1FQ_kIibfMslW84W4SsSCcXS-pojepUBc6f0ehvzig00/template/preview)

A **problem statement** is a clear description of a user need that should be addressed. A **hypothesis statement** communicates your best educated guess on what you think the solution might be to the problem described by your problem statement.
